== 等于操作符 會先強制轉換型別與值 JavaScript中存在隐式转换 規則: 字符串和布尔值都会转换成数值，再比较 用类型，则比较它们是否指向同一个对象 null 和 undefined 相等 === 全等操作符-类型相同，值也需相同 不會強制轉換型別與值 兩個情況例外: Object.is 派上用場的時候 +0 === -0; // true NaN === NaN; // false 除了在比较对象属性为null或者undefined的情况下，我们可以使用相等操作符（==），其他情况建议一律使用全等操作符（===） Object.is 同值比較 (same-value equality) console.log(Object.is(+0, -0)); // false console.log(Object.is(NaN, NaN)); // true (分辨 NaN，可以選擇用 Number.isNaN)
